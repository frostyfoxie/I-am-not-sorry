# I AM NOT SORRY 

> **"Bulbule phodna hi aazaadi hai."** 
> *Popping the spheres is the only way to be free.*

An immersive, interactive digital experience exploring the invisible spheres of societal pressure, body policing, and the weight of patriarchy. This zine serves as both a critique and a celebration of liberation.

[**Explore the Live Zine**](https://frostyfoxie.github.io/I-am-not-sorry/)

---

## 🎨 Artistic Concept
In this digital experience, the user is pursued by a floating, iridescent bubble. This bubble represents the "Log Kya Kahenge" (What will people say?) sphere—it is beautiful yet restrictive, transparent yet confining. 

Through vertical storytelling and interactive physics, the zine navigates through:
- **Pitrasatta (Patriarchy):** The inherited weight of silence.
- **Sanskaar (Stereotypes):** The labels that act as cages.
- **Maryada (Body Policing):** The public auditing of the private self.
- **Insaaf (Justice):** The shift from victim-blaming to empowerment.

---

## 🛠️ Tech Stack & Interactive Features
This project was built to push the boundaries of 2026 web performance and creative coding.

- **Engine:** GSAP (GreenSock Animation Platform) for high-performance scroll triggers and timeline manipulation.
- **Physics:** Custom-built JavaScript "Bubble Engine" with variable dragging, inertia, and stretch-distort logic.
- **Styling:** Tailwind CSS for a modern, mobile-first fluid layout.
- **Typography:** Archivo (Mega headers) & Playfair Display (Serif elegance) via Google Fonts.
- **Interactivity:**
  - **Fluid Cursor:** A mix-blend-mode custom pointer.
  - **Dynamic Refraction:** SVG filters applied to live backdrop-filters to simulate bubble optics.
  - **The "Hope Mode" Toggle:** A site-wide logic swap that changes the narrative and UI upon popping the bubble.
